# objecttrack



This project is the front end of Object Track project based on React/Next.js

## Introduction

Object Track project is a mobile application tracking objects with a supabase backend , a front end Web application as an admin panel as support projects.
This project is the Web application.



## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```
